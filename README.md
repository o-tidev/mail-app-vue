# vue-3-demo

## Project setup
```
yarn install
npm install -g json-server
json-server --watch db.json --port 8000
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
